This repository includes all the scripts required for fully replicating the relevant study.
The SNP_level file includes scripts for the QC steps and identification of initial d-pleiotropic SNPs.
The Loci_level file includes scripts for the clumping algorithm, fine-mapping with SuSiE, and colocalization with coloc.
The gene-level file includes scripts for the positional and eQTL mapping.
The complementary scripts are the scripts with no relevance to the main analyses, but used for producing certain supplementary tables and figures.
Please see the requirements.txt for the necessary libraries and version information.
