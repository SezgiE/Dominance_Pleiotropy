This repository includes all the scripts required for fully replicating the relevant study. 

* **`SNP_level`**: Includes scripts for the QC steps and identification of initial d-pleiotropic SNPs.
* **`Loci_level`**: Includes scripts for the clumping algorithm, fine-mapping with SuSiE, and colocalization with coloc.
* **`gene-level`**: Includes scripts for the positional and eQTL mapping.
* **`Complementary scripts`**: Includes scripts with no relevance to the main analyses, but used for producing certain supplementary tables and figures.

Please see `requirements.txt` for the necessary libraries and version information.
